# Internationalization

Write a function that takes an ISO 639-1 language code and returns a greeting in that language. You can take the examples below or add whatever languages you like.

```js
greet('en'); // 'Hi!'
greet('fr'); // 'Salut!'
greet('pt'); // 'Ol√°!'
greet('de'); // 'Hallo!'
greet('sv'); // 'Hej!'
greet('af'); // 'Haai!'
```

**Answer:**

```js
function greet(langCode) {
    switch (langCode) {
        case 'en':
            return "Hi!";
            break;
        case 'fr':
            return "Salut!";
            break;
        case 'pt':
            return "Ola!";
            break;
        case 'de':
            return "Hallo!";
            break;
        case 'sv':
            return "Hej!";
            break;
        case 'of':
            return "Haai!";
            break;
        default: 
            return "Hi!";   
    }
}
```
